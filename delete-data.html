이 페이지는 제출 전 데이터 삭제 요청 경로 placeholder입니다.

현재 preview

현재 정적 MVP는 서버 데이터가 없고 브라우저 localStorage만 사용합니다. 앱 안의 개인정보 화면에서 로컬 데이터 지우기를 누르면 로컬 삭제가 됩니다.

production 연결 후

Toss login 또는 Supabase가 연결되면 로컬 삭제와 서버 데이터 삭제는 분리해야 합니다. 제출 전 실제 데이터 삭제 요청 URL인 PUBLIC_DATA_DELETION_URL을 연결하고, 서버 데이터 삭제 접수/처리/완료 응답을 별도로 제공해야 합니다.

English

Local reset and server data deletion are separate. A public data deletion request URL must be connected before release.
