함모여 데이터 삭제 요청

이 페이지는 함모여 데이터 삭제 요청 경로 확인 페이지입니다.

현재 앱 화면

현재 GitHub Pages 앱 화면은 방장 기기 브라우저에 만든 모임 상태를 저장합니다. 일반 공유 링크에는 모임 제목과 후보 시간이 공개 스냅샷으로 포함될 수 있습니다. 앱 안의 개인정보 화면에서 로컬 데이터 지우기를 누르면 이 브라우저의 로컬 삭제가 됩니다.

계정 동기화 연결 후

Toss login/account sync가 켜지면 로컬 삭제와 서버 데이터 삭제는 분리해야 합니다. 공개 데이터 삭제 요청 URL은 정식 제출 전 확정해야 하며, 서버 데이터 삭제 접수/처리/완료 응답을 별도로 제공해야 합니다.

English

Local reset and server data deletion are separate. General share links may include a public room snapshot. A public data deletion request URL must be connected before release.
