정적 MVP preview 전용

현재 MVP 범위

이 문서는 함모여 정적 MVP preview에 적용됩니다. 현재 MVP는 Toss login, Supabase, AI API, 광고, 결제, 외부 분석 SDK를 호출하지 않습니다.

처리하는 데이터

  • 방 제목
  • 후보 날짜, 시간, 장소 또는 메모
  • 참여자가 입력한 이름 또는 별명
  • 후보 시간별 선호 응답

저장 위치와 보관 기간

위 데이터는 사용자의 현재 브라우저 localStorage에만 저장됩니다. 서버로 전송하지 않으며, 사용자가 앱의 로컬 데이터 삭제 버튼을 누르거나 브라우저 저장소를 삭제할 때까지 보관됩니다.

삭제 방법

함모여 MVP 화면의 개인정보 메뉴에서 로컬 데이터 지우기를 누르면 현재 브라우저에 저장된 MVP 데이터가 삭제됩니다.

현재 preview에는 계정 생성이나 서버 저장이 없어서 계정 삭제 요청 대상이 없습니다. 향후 Toss login, Supabase, Google Play 계정 흐름을 연결하면 앱 안의 로컬 삭제와 별도의 계정/서버 데이터 삭제 요청 경로를 제공해야 합니다.

제3자 제공과 외부 전송

현재 MVP는 개인정보를 제3자에게 제공하지 않고, 외부 서버나 AI 모델로 전송하지 않습니다. 향후 Toss login, Supabase, AI 문구 다듬기, 공유 링크 기능을 연결할 경우 이 방침과 앱 내 고지를 업데이트해야 합니다.

English Summary

This static MVP preview stores meetup titles, candidate times, nicknames, and availability responses only in this browser's localStorage. It does not call Toss login, Supabase, AI APIs, ads, payments, analytics SDKs, or external servers.

You can remove preview data from the Privacy screen by choosing Clear local data. There is no account deletion flow yet because this preview does not create an account or save server-side user data. Public contact email, external privacy-policy URL, and data-deletion request URL must be finalized before Apps in Toss or Google Play submission.
